# 🔥 FitTransform 12 — 12-Week Fitness Transformation Planner

Two complete **60-minute daily workout plans** built as standalone HTML files. No app install needed — works in any browser and can be installed on your phone as a PWA (Progressive Web App).

> **Goal:** Reduce side fat, lower belly fat and love handles through a structured combination of Strength, Core and Cardio every single day.

## 📲 Install as Mobile App (PWA)

### Android (Chrome)
1. Open the GitHub Pages URL in **Chrome**
2. Tap the **⋮ menu** → **Add to Home Screen**
3. Tap **Install** — appears on your home screen like a native app
4. Works fully **offline** after first load

### iOS (Safari)
1. Open the URL in **Safari**
2. Tap the **Share** button → **Add to Home Screen**
3. Tap **Add** — done ✅

---

## 🛠️ Convert to APK (Android Package)

### Option A — AppsGeyser (Easiest, Free)
1. Go to **[appsgeyser.com](https://appsgeyser.com)**
2. Choose **Website → App**
3. Paste your GitHub Pages URL
4. Download APK → install on phone

### Option B — WebIntoApp
1. Go to **[webintoapp.com](https://webintoapp.com)**
2. Paste URL → choose Android → Download APK

### Option C — PWABuilder
1. Go to **[pwabuilder.com](https://pwabuilder.com)**
2. Paste GitHub Pages URL → Package for stores → Android

## 🚀 How to Use

1. Open `index.html` (V1) or `fitness-plan-v2.html` (V2) in any browser
2. Select your **Week** (W1–W12) using the week pills
3. Tap a **Day tab** (Mon–Sun) to see that day's full 60-min plan
4. Tap any **exercise name** to watch a YouTube form tutorial
5. Check off each exercise as you complete it
6. Tap **Mark day as complete** when done
7. Log your **measurements** weekly in the Progress tab
8. Watch your transformation in the **trend charts**
9. Hit **💾 Save** to persist all progress to localStorage

## 💡 Key Tips for Side & Belly Fat

1. **Core 3× per week** (V2) or every session (V1) — planks, Russian twists, leg raises directly hit obliques and lower abs
2. **Cardio every day** (V2) — HIIT keeps metabolism elevated for 12+ hours after the session
3. **10–12 hour eating window** — finish dinner by 9 PM, breakfast by 7 AM to burn stored fat overnight
4. **120–150g protein daily** — eggs, paneer, dal, Greek yogurt — keeps you full and preserves muscle
5. **3–4L water daily** — reduces bloating and flushes sodium
6. **7–9 hours sleep** — poor sleep spikes cortisol → direct belly fat storage
7. **Friday Full Body** (V2) — squats and deadlifts are the single best fat-burning exercises you can do
